Lecția 1- Structura unui site web

Inițialele HTML provin de la HyperText Markup Language. HyperText se caracterizează prin faptul că un text, imagine, poate fi utilizat ca o referință către un alt document, mecanism prin care se face legatura cu mai multe documente sau cu același document HTML descrie formatul primar în care documentele sunt distribuite și văzute pe Web. HTML este diferit de celelalte limbaje de programare precum C++, Pascal, fiind format din etichete. Acum că te-am lamurit cum stă treaba cu un site, te vom învăța care este structura unui document HTML. Un document HTML este format din mai multe etichete (taguri).

Etichetele sunt de două tipuri:


-pereche: < html > < /html>; <head> </head>; <body> </body>; etc.


-singulare: <br>; <hr>; etc.



Aceasta este structura unui document HTML:


Acum sa explicam fiecare eticheta:


<html> -cu această etichetă începe orice document HTML. Astfel, îi spunem browser-ului că este vorba de un fișier HTML pentru a îl putea afișa.


</html> -aceasta este eticheta de încheiere.



<head> -între aceste etichete se află eticheta <title> <style> și alte etichete specifice acestui tag.


</head> -aceasta este eticheta de încheiere.



<title> -între această etichetă se pune titlul documentului care va aparea in bara de titlu.


</title> -aceasta este eticheta de încheiere.



<body> -tot ce vei scrie între acestă etichetă, va fi afișat in browser


</body> -aceasta este eticheta de încheiere.



Pentru a înțelege mai bine, gândește-te la documentul HTML ca la o pagină într-o carte. Cum iți dai seama că este o pagină dintr-o carte? Simplu, datorită formei, texturii, și informațiilor pe care le conține. Pentru ca internetul să iși poată da seama că un document este o pagină dintr-un site, trebuie ca respectivul document sa îndeplinească două condiții. Prima condiție este ca documentul să înceapă cu eticheta <html> și să se încheie </html>. Cea de-a doua condiție este ca documentul să fie salvat cu extensia ".html".


Dar o pagină dintr-o carte mai are și o porțiune liberă urmată de titlu. În documentul HTML, porțiunea liberă este numită Header. Acest Header se introduce prin eticheta <head>. Tot ce vei scrie între etichetele <head> </head> NU va apărea pe pagină. Între aceste etichete se introduce eticheta <title> care va da numele paginii respective.


Ce mai conține o pagină dintr-o carte? Conținutul. Pentru a introduce conținut într-un document HTML, trebuie să apelăm la eticheta <body>.Tot ce vei scrie în această etichetă va apărea pe pagina site-ului.
